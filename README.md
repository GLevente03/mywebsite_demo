#Levi Around The World – Web Application

Egy saját fejlesztésű, PHP alapú webalkalmazás, amely felhasználói regisztrációt, bejelentkezést és személyre szabott tartalomkezelést biztosít. A projekt célja egy skálázható, biztonságos és modern webes rendszer kialakítása.

🚀 Funkciók
🔐 Felhasználói regisztráció és bejelentkezés
📧 Email alapú megerősítés (tokenes verifikáció)
🌐 Többnyelvű támogatás (session + cookie alapú)
🛡️ Jelszó validáció (erősségi követelményekkel)
🚫 Spam védelem (IP alapú regisztráció limit)
🧹 Automatikus tisztítás nem megerősített felhasználókra
📬 SMTP email küldés (PHPMailer)
🛠️ Használt technológiák

Backend: PHP (OOP, MVC szemlélet)
Adatbázis: MySQL
Email küldés: PHPMailer
Környezet kezelés: vlucas/phpdotenv
Frontend: HTML, CSS
Szerver: Apache (Raspberry Pi)
